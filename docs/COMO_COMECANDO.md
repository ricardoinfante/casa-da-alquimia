## ✅ Status: Pronto para Uso

## 🎯 O Que Você Tem

Um site institucional moderno para a Casa da Alquimia com:

- ✨ Design visual sagrado/natural (azuis, verdes e tons terrosos)
- 📸 Galeria de memórias (fotos e vídeos)
- 🌟 Depoimentos de caminhantes
- 🎵 Player Spotify integrado
- 📱 Responsivo em qualquer dispositivo
- 📲 Links para Instagram, YouTube e WhatsApp

---

```
src/components/
├── Hero.tsx              — Banner principal
```

## 🚀 Como Começar (5 minutos)

### PASSO 1: Rodar o Projeto

```bash
npm install
npm run dev
# Acessar: http://localhost:8086
```

### PASSO 2: Explorar o Site

```
Início | Sobre | Rituais | Memórias | Depoimentos | Conecte-se | Contato
```

### PASSO 3: Adicionar Conteúdo

Para adicionar fotos à galeria, edite `src/components/MemoriasGallery.tsx`.

Para alterar textos, edite os respectivos componentes em `src/components/`.
