# 🎊 Resumo Executivo - Casa da Alquimia v2

## 📌 O Que é Este Projeto

**Casa da Alquimia** é um site institucional para um espaço de prática espiritual e holística. É uma single-page application (SPA) moderna com design visual sagrado/natural.

---

## 🎯 Seções do Site

| Seção | Descrição |
|-------|-----------|
| **Hero** | Banner principal com chamada à ação |
| **Sobre** | Apresentação do espaço |
| **Rituais** | Apresentação das práticas e rituais |
| **Memórias** | Galeria de fotos e vídeos |
| **Depoimentos** | Palavras de quem viveu a transformação |
| **Conecte-se** | Feeds Instagram e YouTube |
| **Apoiar** | Seção de doações |
| **Contato** | Formulário de contato |

---

## 🎨 Design

O site usa uma paleta sagrada/natural:
- Azuis cobalto e profundos para elementos primários
- Verdes musgo e verde-água para fundos
- Tons terrosos (terracota, âmbar) para acentos e labels
- Tipografia Cinzel para títulos, Lato para corpo

---

## 📊 Dados Técnicos

| Aspecto | Detalhes |
|---------|---------|
| **Framework** | React 18 + TypeScript |
| **Estilo** | Tailwind CSS + Shadcn/UI |
| **Build** | Vite |
| **Backend** | Supabase (PostgreSQL + Storage) |
| **Deploy** | Plesk (build estático) |

---

## 📱 Compatibilidade

✅ **Desktop** — Layout completo
✅ **Tablet** — Layout otimizado
✅ **Mobile** — Layout responsivo
✅ **Acessibilidade** — Alt text, keyboard navigation

---

## 🚀 Como Começar

```bash
npm install
npm run dev
# Acessar: http://localhost:8086
```

Para fazer deploy:
```bash
npm run build
# Enviar pasta dist/ para o servidor Plesk
```
